# NCSS 正式抓取记录（2026-04-16）

这份记录对应项目第一版 `NCSS` 均衡样本抓取，目的是在较短时间内先形成一份可直接分析的招聘文本数据。

## 1. 第一版抓取范围

- 城市：`北京`、`上海`、`广州`、`深圳`、`武汉`、`成都`
- 关键词：`Python`、`数据分析`、`运营`、`行政`、`会计`、`产品经理`
- 每个“城市 × 关键词”组合最多抓取 `3` 页列表
- 列表接口每页 `20` 条

第一版配置文件：

- `data/input/ncss/ncss_batch_config.json`

## 2. 使用的主命令

### 抓取列表并导出详情种子

```bash
python3 src/platforms/ncss/fetch_ncss_jobs.py \
  --config data/input/ncss/ncss_batch_config.json \
  --list-output data/raw/ncss/records/ncss_listings_raw.jsonl \
  --detail-seed-output data/input/ncss/ncss_detail_urls_generated.csv \
  --summary-output data/processed/ncss/ncss_list_query_summary.csv \
  --max-jobs 600
```

结果：

- 列表原始记录：`2080`
- 去重后唯一职位：`2074`
- 初始详情页种子：`600`

### 构造均衡详情种子

```bash
python3 src/platforms/ncss/build_balanced_ncss_detail_seeds.py \
  --input data/raw/ncss/records/ncss_listings_raw.jsonl \
  --output data/input/ncss/ncss_detail_urls_balanced.csv \
  --max-jobs 600
```

### 抓取详情页 HTML

```bash
python3 src/platforms/ncss/fetch_pages.py \
  --seed-file data/input/ncss/ncss_detail_urls_balanced.csv \
  --manifest data/raw/ncss/manifests/ncss_detail_manifest_balanced.jsonl \
  --output-dir data/raw/ncss/html \
  --config data/input/ncss/platform_ncss_detail.json \
  --delay 0.2 \
  --skip-existing
```

### 解析详情页

```bash
python3 src/platforms/ncss/parse_details.py \
  --manifest data/raw/ncss/manifests/ncss_detail_manifest_balanced.jsonl \
  --config data/input/ncss/platform_ncss_detail.json \
  --output data/raw/ncss/records/ncss_jobs_balanced_raw.jsonl \
  --overwrite
```

### 清洗结果表

```bash
python3 src/clean_jobs.py \
  --input data/raw/ncss/records/ncss_jobs_balanced_raw.jsonl \
  --output data/processed/ncss/ncss_jobs_balanced_clean.csv
```
