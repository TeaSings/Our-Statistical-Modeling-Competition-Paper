## 维护原则

- `docs/` 只放项目文档
- 论文原文统一放到 `papers/reference_library/`
- 与数据直接对应的文件说明，优先写到 `data/` 各子目录的 `README.md`
- 带日期的抓取记录保留为过程档案，最终可交付状态以 `NCSS全量数据核验与交付说明-2026-04-16.md` 为准
