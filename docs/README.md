# docs 说明

这个目录只存放项目文档，不放代码、原始数据和论文 PDF。

## 建议先读什么

- 看当前是否已经可以交付：先读 `NCSS全量数据核验与交付说明-2026-04-16.md`
- 看数据源和脚本入口：再读 `中文数据来源与脚本说明.md`
- 看历史抓取过程：最后读两份带日期的 NCSS 记录

## 当前文档

| 文件 | 内容 |
| --- | --- |
| `NCSS全量数据核验与交付说明-2026-04-16.md` | 当前全国 NCSS 主数据的核验结果、样本形成链路和提交建议 |
| `中文数据来源与脚本说明.md` | 中文互联网可用数据源、目录结构和脚本入口说明 |
| `NCSS正式抓取记录-2026-04-16.md` | NCSS 第一版均衡样本抓取记录 |
| `NCSS全地区扩展抓取记录-2026-04-16.md` | NCSS 全地区列表扩展、详情补抓和目录重构记录 |
| `数据分析初步方法.md` | 论文思路、变量设计和分析框架笔记 |

## 维护原则

- `docs/` 只放文本型项目文档
- 论文原文统一放在 `papers/pdfs/`
- 与数据目录、脚本目录直接对应的说明优先写在各自的 `README.md`
- 带日期的抓取记录属于过程档案；如果与其他说明存在时间差，以核验文档和最新 README 为准
