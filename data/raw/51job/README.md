# data/raw/51job 说明

这里存放前程无忧校招专题页抓取后的原始结果。

## 子目录

- `records/`：标准化前的职位原始 JSONL
- `manifests/`：按专题页记录的抓取 manifest
- `html/pages/`：专题页 HTML 快照
- `html/assets/`：专题页本地 JS / HTML 资产快照

## 当前主用文件

- `records/51job_campus_jobs_raw.jsonl`：当前 51job 校招原始职位记录
- `manifests/51job_campus_seed_manifest.jsonl`：按专题页记录的处理结果与策略
