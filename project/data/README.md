# Project Data Outputs

- `raw/`: original live crawl CSV outputs from `run_batch.py` / `run_pipeline.py`
- `clean/`: first-pass cleaned live CSV outputs
- `clean_v2/`: recleaned live CSV outputs
- `final/seen_jobs.txt`: incremental dedupe state used by the live crawl
- `final/master_jobs.csv`: research-ready merged file generated by `python project/merge_current_outputs.py`
- `jobs.csv` / `jobs.db`: live crawl storage outputs

Historical archive-crawl outputs were removed on purpose. The research workflow is now:

1. keep collecting current live data with `project/run_pipeline.py`
2. merge current outputs into `project/data/final/master_jobs.csv`
3. compare this live dataset with historical data or published results from other papers
